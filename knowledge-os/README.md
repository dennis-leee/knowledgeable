# Knowledge OS

**AI зҹҘиҜҶеӣҫи°ұиҮӘз”ҹй•ҝзі»з»ҹ** - иҮӘеҠЁд»Һ URL жҸҗеҸ–зҹҘиҜҶгҖҒжһ„е»әзҹҘиҜҶеӣҫи°ұгҖҒз”ҹжҲҗ AI еҸҜз”Ёзҡ„ Skills/MCP ж јејҸгҖӮ

## зү№жҖ§

- **иҮӘеҠЁзҹҘиҜҶжҸҗеҸ–**: д»Һд»»ж„Ҹ URL иҮӘеҠЁжҸҗеҸ–е®һдҪ“гҖҒе…ізі»гҖҒжҙһеҜҹ
- **зҹҘиҜҶеӣҫи°ұжһ„е»ә**: иҮӘеҠЁжһ„е»әеҸҜжҹҘиҜўзҡ„зҹҘиҜҶеӣҫи°ұ
- **еӨҡж јејҸиҫ“еҮә**: ж”ҜжҢҒ MarkdownгҖҒJSON GraphгҖҒVector Embeddings
- **AI Skills з”ҹжҲҗ**: иҫ“еҮә AI Agent еҸҜзӣҙжҺҘдҪҝз”Ёзҡ„ Skills ж јејҸ
- **MCP е…је®№**: ж”ҜжҢҒ Model Context Protocol ж ҮеҮҶ
- **иҮӘжҲ‘дҝ®еӨҚ**: Validation + LLM Repair иҮӘеҠЁдҝ®еӨҚж•°жҚ®иҙЁйҮҸй—®йўҳ
- **дәәзұ»д»Ӣе…Ҙ**: д»…еңЁдҪҺзҪ®дҝЎеәҰж—¶и§ҰеҸ‘дәәе·Ҙе®Ўж ё
